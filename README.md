
# Il Pendolare e il Pensiero
Un progetto interattivo per pendolari, basato su enigmi filosofici e quiz.

## Struttura
- index.html → Home
- chapters/ → Capitoli con quiz
- assets/css → Stili
- assets/js → Script
- data/questions.json → Domande dinamiche

## Come pubblicare su GitHub Pages
1. Crea un repository su GitHub.
2. Carica tutti i file.
3. Vai su Settings → Pages → Seleziona branch `main` e cartella `/root`.
4. Copia il link del sito e genera QR code per le pagine.

## Come aggiungere nuovi capitoli
- Duplica un file in `chapters/`.
- Aggiungi la domanda in `questions.json`.
- Aggiorna il JS se necessario.
